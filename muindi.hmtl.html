Create a moded bot for your WhatsApp as follows.

Deploy on Heroku

  1. Click SCAN and scan the QR code through the "WhatsApp Linked Devices" option in your WhatsApp app.
  2. You will get a session ID in WhatsApp, copy the ID only.
  3. If you don't have an account on Heroku, create an account now: Heroku.
  4. If you don't have a GitHub account, sign up now: GitHub.
  5. FORK this repository.
  6. Now DEPLOY.
